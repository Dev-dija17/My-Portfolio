# Assets Folder

## Profile Picture Instructions

To add your profile picture to the portfolio:

1. **Add your photo** to this `assets` folder
2. **Name it** `profile-picture.jpg` (or update the path in index.html)
3. **Recommended specs:**
   - Format: JPG, PNG, or WebP
   - Size: At least 400x400 pixels (square format works best)
   - File size: Under 1MB for fast loading

## Project Images Instructions

Add screenshots of your projects to showcase your work:

1. **E-Commerce Project**: Replace `project1-ecommerce.jpg`
2. **Task Manager**: Replace `project2-taskmanager.jpg`  
3. **Weather Dashboard**: Replace `project3-weather.jpg`

**Recommended specs for project images:**
- Format: JPG, PNG, or WebP
- Size: 800x400 pixels (2:1 aspect ratio works best)
- File size: Under 500KB each
- High quality screenshots showing your project's interface

## Alternative File Names:
If you prefer different filenames, update the `src` attributes in index.html:
```html
<img src="assets/your-project-name.jpg" alt="Project Name" class="project-img">
```
